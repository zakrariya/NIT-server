# Task 1 - Create a LinkedIn Account

## Step 1
Open the LinkedIn website:

```text
https://www.linkedin.com
```

## Step 2
Click:

```text
Join now
```

## Step 3
Enter:

- Email address
- Password

Then click:

```text
Agree & Join
```

## Step 4
Enter:

- First Name
- Last Name

Click:

```text
Continue
```

## Step 5
Verify your email address if LinkedIn asks for verification.

## Step 6
Complete the basic setup steps shown by LinkedIn.
